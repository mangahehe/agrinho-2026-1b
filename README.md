# Projeto Agrinho 2026 - Agricultura Familiar 🚜🌱

Este site foi desenvolvido como parte do projeto para o **Concurso Agrinho 2026** do Paraná. O tema central é a **Agricultura Familiar**, explorando sua importância socioeconômica, práticas sustentáveis e o papel fundamental que desempenha no estado do Paraná.

## 📝 Tema do Ano
> **"Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente"**

## 🚀 Como Executar o Projeto

Este é um projeto web estático (HTML, CSS e JavaScript puro). Para visualizá-lo:

1. **Baixe os arquivos:** Certifique-se de ter os arquivos `index.html`, `style.css` e `script.js` na mesma pasta.
2. **Abra no Navegador:** Dê um clique duplo no arquivo `index.html` ou arraste-o para uma aba do seu navegador (Chrome, Edge, Firefox, etc.).
3. **No VSCode (Recomendado):**
   - Instale a extensão **Live Server**.
   - Clique com o botão direito no `index.html` e selecione **"Open with Live Server"**. Isso permite ver as alterações em tempo real enquanto você edita o código.

## 📂 Estrutura de Arquivos

- `index.html`: Contém toda a estrutura do site e o conteúdo informativo detalhado sobre agricultura familiar.
- `style.css`: Responsável por todo o visual, cores, fontes e garantir que o site funcione bem em celulares (responsividade).
- `script.js`: Adiciona interatividade, como o menu móvel, animações de números e validação do formulário.
- `conteudo.md`: Um guia de referência com os dados e textos utilizados no site.

## ✨ Funcionalidades do Site

- **Design Responsivo:** O site se adapta a computadores, tablets e smartphones.
- **Animações de Scroll:** Elementos aparecem suavemente conforme você navega pela página.
- **Contadores Animados:** Estatísticas que ganham vida quando entram na tela.
- **Navegação Suave:** Links que deslizam suavemente até a seção desejada.
- **Formulário Interativo:** Simulação de contato com validação de campos.

## 💡 Dicas para o Aluno (Agrinho 2026)

- **Personalização:** No `index.html`, procure a seção de contato ou o rodapé e adicione o nome da sua escola e sua cidade.
- **Imagens:** O site utiliza ícones (FontAwesome) e cores sólidas. Você pode enriquecer o projeto adicionando fotos reais de propriedades rurais da sua região usando a tag `<img src="caminho/da/foto.jpg">`.
- **Apresentação:** Ao apresentar o site para os avaliadores, destaque como a agricultura familiar une a tradição com a preservação ambiental (o equilíbrio pedido no tema).
